Qualquer Coisa

Em fase de desenvolvimento a aplicação móvel "Qualquer Coisa" que tem como objetivo ajudar as pessoas a lidar com o problema comum de não saber o que cozinhar. A aplicação vai fornecer uma ementa semanal aleatória, com receitas da sua base de dados, para almoço e jantar, bem como uma lista de ingredientes necessários para as receitas. Os utilizadores poderão pesquisar por receitas específicas ou por tipo de refeição, bem como ter acesso a três a quatro receitas aleatórias diariamente. A aplicação também terá a opção de definir restrições alimentares e ligar o utilizador ao seu nutricionista. Com a opção de criar receitas personalizadas e uma lista de receitas favoritas, a aplicação será um guia útil para qualquer pessoa que queira cozinhar de forma mais fácil e eficiente.

Apresentação da Aplicação

O meu projeto é uma aplicação chamada "Qualquer Coisa", que tem como objetivo ajudar a combater o problema de muitas pessoas que muitas vezes não sabem o que cozinhar. Através da criação de uma ementa semanal, a aplicação irá buscar receitas de forma aleatória a uma base de dados para todos os almoços e jantares da semana. Ao final da semana, a aplicação irá atualizar a ementa para a semana seguinte e adicionar todos os ingredientes necessários para essas ementas ao ecrã da "Loja".

Além disso, a aplicação contará com uma função de pesquisa, onde o utilizador poderá pesquisar por uma receita específica que queira confecionar ou então pesquisar por tipo de refeição, como almoço ou sobremesa. Outra função interessante será um espaço onde o utilizador possa esporadicamente encontrar 3 a 4 receitas aleatórias para escolher a que mais lhe agrada para confecionar, similar a uma sugestão diária de um restaurante.

Para aqueles que possuem restrições alimentares, a aplicação terá uma função para definir essas restrições, como só ser vegano ou vegetariano, que será usado para filtrar as receitas. A aplicação também terá uma função para conectar o utilizador ao seu nutricionista, permitindo que o nutricionista faça uma melhor ementa e adicione restrições alimentares específicas para o paciente.

Outras funções interessantes serão um espaço onde o utilizador poderá criar suas próprias receitas, além de uma lista de favoritos para armazenar receitas recomendadas ou pesquisadas que lhe agradaram.

Estamos entusiasmados com o desenvolvimento deste aplicativo e esperamos que ele possa ajudar muitas pessoas a ter uma alimentação mais saudável e variada, mesmo quando falta inspiração ou tempo para cozinhar.
